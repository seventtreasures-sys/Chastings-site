# Chastings Co. Official Website

A digital recruitment and informational hub for The Barony of Chastings RP Guild, built with a focus on modern design, accessibility, and high performance. This is the initial static version of the site, showcasing the main product line and brand identity.

Responsive Design: Optimized for seamless viewing across desktop, tablet, and mobile devices.

Clean, Modern UI: A sleek, user-centric interface.

Information Showcase: Dedicated sections for various Chastings subjects (how to join, about us, etc).

Efficient Codebase: Built exclusively with semantic HTML5 and clean, organized CSS3.

HTML5: For the core structure and semantic content.

CSS3: For styling and layout.

To get a copy of this project up and running on your local machine for development and testing purposes, follow these simple steps.

Prerequisites:
You only need a modern web browser (like Chrome, Firefox, or Edge) to view the project.
